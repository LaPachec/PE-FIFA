# Requisitos

## Visao geral

O FIFA Tournament Manager deve permitir que amigos organizem campeonatos de FIFA/EA FC com participantes, partidas, fases e resultados centralizados.

## Requisitos funcionais iniciais

1. Criar usuarios.
2. Criar campeonatos.
3. Cadastrar participantes em campeonatos.
4. Registrar partidas e placares.
5. Consultar status de um campeonato.
6. Preparar suporte para liga, mata-mata e formato misto.

## Requisitos nao funcionais

1. Usar TypeScript em todos os apps e packages.
2. Centralizar regras de negocio na API.
3. Manter o front-end sem regras complexas de campeonato.
4. Usar PostgreSQL como banco relacional.
5. Usar Prisma para acesso ao banco.
6. Manter arquitetura simples e pronta para crescimento.

## Fora do escopo desta etapa

1. Autenticacao real.
2. CRUD completo.
3. Geracao automatica de partidas.
4. Calculo de classificacao.
5. Regras avancadas de desempate.
